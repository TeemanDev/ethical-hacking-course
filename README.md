# Ethical Hacking Course

A complete platform for ethical hacking training with:
- Course registration
- Crypto payment verification
- Student dashboard
- Download tracking
- Admin panel

## Features
- ✅ User registration and login
- ✅ Crypto payments (Bitcoin, Ethereum, USDC)
- ✅ Resource downloads
- ✅ Student progress tracking
- ✅ Admin dashboard

## Tech Stack
- Node.js + Express
- SQLite database
- HTML/CSS/JavaScript frontend
